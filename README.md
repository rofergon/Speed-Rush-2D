## üéÆ Main Features

### üöó NFT Vehicle System
- **AI Generation**: 
  - Each vehicle is uniquely generated by AI
  - Generation system based on customizable prompts
  - Different visual styles: Cartoon, Realistic, Anime
  - Each vehicle is an NFT with unique metadata

### üîß NFT Parts System
- **Interchangeable Parts**:
  - Engine: Affects speed and acceleration
  - Transmission: Influences speed changes and handling
  - Core: Determines vehicle base characteristics
- **Part Features**:
  - Each part is an independent NFT
  - Rarity system and unique statistics
  - Possibility of exchange between vehicles
  - Directly affect race performance

### üèÅ Racing System
- **Game Modes**:
  - Time Trial: Compete for best times and contribute to the prize pool
  - Daily challenges with cumulative pools
  - Special tournaments with guaranteed prizes
- **Game Mechanics**:
  - Arcade topdown 2D physics system
  - Drifting and boost
  - Different track types
  - Lap system and result minting
- **Prize Pool System**:
  - Each result minting contributes to the current tournament pool
  - Automatic prize distribution at tournament end
  - Distribution percentages:
    - 50% for first place
    - 30% for second place
    - 15% for third place
    - 5% for community fund
  - Verifiable smart contract for distribution
  - History of pools and previous winners

    - Buying/selling in the marketplace

    - Participating in special events
- **Marketplace System**:
  - Vehicle and parts listing
  - Offer system
  - Advanced search filters

## üîß Detailed Setup

1. Clone the repository
```bash
git clone https://github.com/rofergon/Speed-Rush-2D
```

2. Install dependencies
```bash
npm install
```

3. Configure environment variables
```env
VITE_RACE_LEADERBOARD_ADDRESS=<contract_address>
VITE_CAR_NFT_ADDRESS=<contract_address>
VITE_PART_NFT_ADDRESS=<contract_address>
VITE_MARKETPLACE_ADDRESS=<contract_address>
VITE_GARAGE_ADDRESS=<contract_address>
VITE_LENS_NETWORK_HUB=<hub_address>
```

4. Start development server
```bash
npm run dev
```
